## 4. วิธีเพิ่มบัญชีครู/นักเรียนใหม่

1. เข้า [supabase.com](https://supabase.com) → เลือกโปรเจกต์ → เมนู **Authentication → Users**
2. กด **Add user** กรอกอีเมล (ต้องเป็นโดเมนโรงเรียน) + รหัสผ่าน → ติ๊ก **Auto Confirm User**
3. ในช่อง **User Metadata** ใส่บทบาท:
   - ครู: `{ "role": "admin" }`
   - นักเรียน: `{ "role": "student" }`
4. กด Create — ผู้ใช้ล็อกอินได้ทันที

## 5. การใช้งานพื้นฐาน

**ฝั่งครู (admin):**
- แท็บ "จัดการกิจกรรม" → เพิ่มกิจกรรม + กด "อัปโหลดเกียรติบัตร" เพื่อใส่รูปเกียรติบัตรเปล่าของแต่ละกิจกรรม
- แท็บ "คำขออนุมัติ" → ดูคำขอ + รูปหลักฐานของนักเรียน → กด "อนุมัติ" (ออกเกียรติบัตรทันที) หรือ "ปฏิเสธ"

**ฝั่งนักเรียน:**
- เลือกกิจกรรม → แนบรูปหลักฐาน → ส่งคำขอ
- เมื่อครูอนุมัติ → ดาวน์โหลด/พิมพ์เกียรติบัตรได้ที่แท็บ "เกียรติบัตรของฉัน"

> ถ้าครูยังไม่อัปโหลดเกียรติบัตรเปล่า ระบบจะออกเกียรติบัตรแบบมาตรฐาน (HTML) ให้อัตโนมัติ
